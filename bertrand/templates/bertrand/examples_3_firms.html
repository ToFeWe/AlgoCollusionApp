Beispiele

Beispiel 1: Firma A setzt einen Preis von 3 Taler, Firma B setzt einen Preis von 3 Taler, Firma C setzt einen Preis von 4 Taler. Somit haben Firma A und B gemeinsam den niedrigsten Preis gesetzt. Firma A und B verkaufen beide die gleiche Anzahl an Produkten. Beide Firmen haben jeweils 30 Kunden und bekommen damit den gleichen Gewinn von 90 Talern. Firma C verkauft nichts und hat einen Gewinn von 0 Taler.

Firma A Firma B Firma C
Preise 3 3 4
Gewinne 90 90 0


Beispiel 2: Firma A setzt einen Preis von 2 Taler, Firma B setzt einen Preis von 2 Taler, Firma C setzt einen Preis von 2 Taler. Somit haben Firma A, B und C gemeinsam den niedrigsten Preis gesetzt. Sie verkaufen alle die gleiche Anzahl an Produkten (jeweils 20) und bekommen damit den gleichen Gewinn von 40 Talern.

Firma A Firma B Firma C
Preise 2 2 2
Gewinne 40 40 40


Beispiel 3: Firma A setzt einen Preis von 1 Taler, Firma B setzt einen Preis von 2 Taler, Firma C setzt einen Preis von 3 Taler. Somit hat Firma A den niedrigsten Preis gesetzt. Firma A verkauft als einzige das Produkt zu einem Preis von 1 Taler an alle 60 Kunden und bekommt damit einen Gewinn von 60 Talern. Firma B und C verkaufen beide nichts und haben einen Gewinn von 0 Taler.

Firma A Firma B Firma C
Preise 1 2 3
Gewinne 60 0 0


Beispiel 4: Firma A setzt einen Preis von 5 Taler, Firma B setzt einen Preis von 5 Taler, Firma C setzt einen Preis von 5 Taler. Somit haben Firma A, B und C gemeinsam den niedrigsten Preis gesetzt. Die Kunden sind jedoch lediglich bereit das Produkt zu einem Preis von 4 Talern zu kaufen. Daher verkauft keine Firma ihre Produkte und alle Firmen bekommen einen Gewinn von 0 Taler.

Firma A Firma B Firma C
Preise 5 5 5
Gewinne 0 0 0
